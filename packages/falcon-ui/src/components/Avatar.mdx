`Avatar` component renders as an `<img>` element and accepts all of the `<img>` props, it's size can be easily adjusted via theme-based `size`,
 but it also accepts other [responsive, theme-based style props](/responsive-props) that allow further styling customizations.

```javascript
import { Avatar } from '@deity/falcon-ui';
```
